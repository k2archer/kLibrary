# 订阅书单信息数据格式

#### 用户信息格式

```
[orders]
	order_id:
	user_id:
	book_id:
	orderd_time:
```

| 字段     | 类型    | 说明      | 字段  | 类型    | 说明   |
| -------- | ------ | -------- | ---- | ------ | ------ |
| order_id | 整数    | 书单编号、自动增长、唯一性 | user_id | 整数 | 用户名 |
| book_id | 整数 | 书籍编号 | ~~orderd_time~~ | 日期 | 订阅日期时间 |
