# 书籍信息数据格式

#### 书籍信息格式

```
[book]
	book_id:
	book_name:
	book_code:
	book_cover:
```

| 字段     | 类型    | 说明      | 字段  | 类型    | 说明   |
| -------- | ------ | -------- | ---- | ------ | ------ |
| book_id | 整数    | 书编号、自动增长、唯一性 | book_name | 字符串  | 书名 |
| book_code | 字符串  | 书号即 ISBN 代码 | book_cover | 字符串 | 书封面图片路径 |
